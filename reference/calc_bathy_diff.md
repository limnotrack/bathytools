# Calculate the difference between observed and predicted bathymetry

Calculate the difference between observed and predicted bathymetry

## Usage

``` r
calc_bathy_diff(obs, pred)
```

## Arguments

- obs:

  SpatRaster object with the observed bathymetry.

- pred:

  SpatRaster object with the predicted bathymetry.

## Value

SpatRaster object with the difference between the observed and predicted
bathymetry.
