# Get Lake Depth from Bathymetry Raster This function calculates the lake depth from a bathymetric raster.

Get Lake Depth from Bathymetry Raster This function calculates the lake
depth from a bathymetric raster.

## Usage

``` r
get_lake_depth(bathy_raster)
```

## Arguments

- bathy_raster:

  SpatRaster object with the bathymetry data.

## Value

numeric. The lake depth.
